## 权限说明

文件权限格式：`drwxrwxrwx`
- 第1位：文件类型（d=目录，-=文件，l=链接）
- 第2-4位：所有者权限（rwx）
- 第5-7位：组权限（rwx）
- 第8-10位：其他用户权限（rwx）

权限含义：
- r（read）：读权限
- w（write）：写权限
- x（execute）：执行权限

## 常用快捷键

- `Ctrl + C`：中断当前命令
- `Ctrl + Z`：暂停当前命令
- `Ctrl + L`：清屏（等同于clear命令）
- `Tab`：自动补全
- `↑/↓`：浏览命令历史
